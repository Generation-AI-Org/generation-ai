# Plan 02-01: Auth Package

## Goal

Erstelle `@genai/auth` Package mit Supabase Client-Wrappern. Das Package bietet:
- Browser-Client (CSR) mit konfigurierbaren Cookie-Options
- Server-Client (SSR, App Router)
- Admin-Client (Service Role)
- Helper-Funktionen (getUser, getSession)

Beide Apps importieren dann von `@genai/auth/*` statt lokalen `lib/supabase/` Modulen.

## Detailed Implementation

### Task 1: Package-Struktur

```bash
# packages/auth/package.json
{
  "name": "@genai/auth",
  "version": "0.0.1",
  "private": true,
  "main": "./src/index.ts",
  "exports": {
    ".": "./src/index.ts",
    "./browser": "./src/browser.ts",
    "./server": "./src/server.ts",
    "./admin": "./src/admin.ts",
    "./helpers": "./src/helpers.ts"
  },
  "dependencies": {
    "@supabase/ssr": "^0.10.2",
    "@supabase/supabase-js": "^2.103.0"
  },
  "peerDependencies": {
    "next": ">=15"
  }
}
```

**Warum Subpath-Exports:** `@genai/auth/server` kann nur in Server Components importiert werden. Verhindert, dass Service Role Key versehentlich im Client-Bundle landet.

### Task 2: Auth-Module

**browser.ts** - Konfigurierbare Cookie-Options:
```typescript
import { createBrowserClient, CookieOptionsWithName } from '@supabase/ssr'

export function createClient(cookieOptions?: CookieOptionsWithName) {
  return createBrowserClient(
    process.env.NEXT_PUBLIC_SUPABASE_URL!,
    process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!,
    cookieOptions ? { cookieOptions } : undefined
  )
}
```

**server.ts** - Next.js App Router kompatibel:
```typescript
import { createServerClient } from '@supabase/ssr'
import { cookies } from 'next/headers'

export async function createClient() {
  const cookieStore = await cookies()
  return createServerClient(
    process.env.NEXT_PUBLIC_SUPABASE_URL!,
    process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!,
    {
      cookies: {
        getAll: () => cookieStore.getAll(),
        setAll: (toSet) => {
          try {
            toSet.forEach(({ name, value, options }) =>
              cookieStore.set(name, value, options)
            )
          } catch { /* Server Components cannot set cookies */ }
        },
      },
    }
  )
}
```

**admin.ts** - Service Role fuer API-Routes:
```typescript
import { createClient } from '@supabase/supabase-js'

export function createAdminClient() {
  return createClient(
    process.env.NEXT_PUBLIC_SUPABASE_URL!,
    process.env.SUPABASE_SERVICE_ROLE_KEY!,
    { auth: { autoRefreshToken: false, persistSession: false } }
  )
}
```

### Task 3: Apps umstellen

**Website Browser-Client** - Keine Cookie-Options (Default-Verhalten):
```typescript
// apps/website/lib/supabase/client.ts
export { createClient } from '@genai/auth/browser'
```

**Tools-App Browser-Client** - Mit Cross-Subdomain Cookies:
```typescript
// apps/tools-app/lib/supabase/browser.ts
import { createClient as createAuthClient } from '@genai/auth/browser'

export function createClient() {
  return createAuthClient({
    domain: '.generation-ai.org',
    path: '/',
    sameSite: 'lax',
    secure: true,
  })
}
```
