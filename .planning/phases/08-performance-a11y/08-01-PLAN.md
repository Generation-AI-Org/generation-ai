  <name>Task 1: Lighthouse CLI Setup und Audit ausfuehren</name>
  <files>.planning/phases/08-performance-a11y/LIGHTHOUSE-AUDIT.md</files>
  <action>
1. Lighthouse CLI installieren falls nicht vorhanden: `pnpm add -D lighthouse -w`

2. Beide Apps im Production-Build lokal starten (oder deployed URLs nutzen):
   - Website: https://generation-ai.org
   - tools-app: https://tools.generation-ai.org

3. Lighthouse CLI ausfuehren fuer beide Apps:
   ```bash
   npx lighthouse https://generation-ai.org --output=json --output-path=.planning/phases/08-performance-a11y/lighthouse-website.json --chrome-flags="--headless"
   npx lighthouse https://tools.generation-ai.org --output=json --output-path=.planning/phases/08-performance-a11y/lighthouse-tools.json --chrome-flags="--headless"
   ```

4. Network-Tab in DevTools pruefen: Verifizieren dass KEINE Requests zu `fonts.googleapis.com` oder `fonts.gstatic.com` gehen.
   - Beide Apps nutzen `next/font/google` welches Fonts lokal hostet
   - Sollte bereits OK sein, aber explizit verifizieren

5. LIGHTHOUSE-AUDIT.md erstellen mit:
   - Datum des Audits
   - Scores beider Apps (Performance, A11y, Best Practices, SEO)
   - Top 3-5 Issues pro Kategorie (falls unter 90)
   - Google Fonts Verifizierung (Third-Party Requests: Ja/Nein)
   - Empfehlungen fuer Fixes
  </action>
