| **Build smoke (Tools)** | `pnpm --filter @genai/tools-app build` |
| **Estimated runtime** | ~60s pro Spec (Playwright), <5s (Vitest) |
| **Dev server (Website)** | `pnpm dev:website` (localhost:3000) |
| **Dev server (Tools)** | `pnpm dev:tools` (localhost:3001) |

---

## Sampling Rate

- **After every task commit:** Run quick command für betroffene Route + relevant build smoke
- **After every plan wave:** Run full Playwright suite für Track + Vitest unit tests
- **Before `/gsd-verify-work`:** Full `pnpm e2e` muss green sein, Build muss `ƒ /events` zeigen (CSP-Pitfall — siehe LEARNINGS.md)
- **Max feedback latency:** 60 seconds
