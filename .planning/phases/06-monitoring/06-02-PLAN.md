<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Client -> Sentry | Error-Daten werden an externes System gesendet |
| Build -> Sentry | Source Maps werden bei Build hochgeladen |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-03 | I (Information Disclosure) | Sentry Client | mitigate | tracesSampleRate=0.1 begrenzt Daten, keine PII in Error Messages |
| T-06-04 | I (Information Disclosure) | Source Maps | mitigate | hideSourceMaps: true verhindert Client-Zugriff auf Source Maps |
| T-06-05 | T (Tampering) | SENTRY_AUTH_TOKEN | mitigate | Token nur in Vercel Env Vars, nicht im Code |
</threat_model>
