# Plan 21-07 — Kontakt Section

> Kontaktbox als abschließende Section der `/about`-Seite. 3 Kontaktwege: Allgemeine Anfragen (Mailto), Partnerschaften (Link `/partner`), Aktiv mitmachen (Anker `#mitmach`). Single-Card mit `divide-y`-Trennern. Anker `id="kontakt"`.

Server/Client-Component, die 3 Kontaktzeilen in einer Card rendert. Mailto für Email, Next.js `<Link>` für interne Route, Anker-Link für same-page-Navigation. Keine dynamische Logik — reine Präsentation.
</objective>
