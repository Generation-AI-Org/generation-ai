1. Replace apps/website/app/sitemap.ts per RESEARCH §7.1:

```typescript
import type { MetadataRoute } from "next";
import { getAllArticles } from "@/lib/mdx/community";

const BASE = "https://generation-ai.org";

export default async function sitemap(): Promise<MetadataRoute.Sitemap> {
  const articles = await getAllArticles();

  const articleEntries: MetadataRoute.Sitemap = articles.map((a) => ({
    url: `${BASE}/community/artikel/${a.slug}`,
    lastModified: new Date(a.frontmatter.date),
    changeFrequency: "monthly",
    priority: 0.7,
  }));

  return [
    {
      url: BASE,
      lastModified: new Date(),
      changeFrequency: "weekly",
      priority: 1,
    },
    {
      url: `${BASE}/community`,
      lastModified: new Date(),
      changeFrequency: "weekly",
      priority: 0.9,
    },
    ...articleEntries,
  ];
}
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| MDX frontmatter → JSON-LD `<script>` tag | Frontmatter is editor-controlled MDX. XSS-escape via `\u003c` mandatory per Next.js doc |
| `[slug]` URL param → `import()` | Bound by `dynamicParams=false` + `generateStaticParams` whitelist — no path traversal possible |
| circleUrl → external `<a>` | Editor-controlled MDX, target=_blank + rel=noopener mandatory |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-26-03-01 | Tampering | JSON-LD script with frontmatter title containing `<` | mitigate | `JSON.stringify(json).replace(/</g, "\\u003c")` — Pflicht-Pattern, tested in schema.test.ts XSS case |
| T-26-03-02 | Tampering | `[slug]` path-traversal via `import(@/content/community/${slug}.mdx)` | mitigate | `dynamicParams = false` + `generateStaticParams` whitelist; unknown slug returns notFound() before import() |
| T-26-03-03 | Information Disclosure | 500-error leaks fs-path on read failure | mitigate | Plan 26-01 throws with file-path context; Next.js wraps to generic 500. Production logs receive full error. Acceptable. |
| T-26-03-04 | Tampering | open-redirect via circleUrl | accept | D-14 trust-the-author. Editor-controlled MDX, no user-input path. README documents. |
