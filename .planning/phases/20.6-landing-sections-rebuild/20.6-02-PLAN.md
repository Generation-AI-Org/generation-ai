# Plan 20.6-02 — Offering Polish

> Hero-Level Polish der Offering-Section (4-Card BentoGrid: Community / Wissensplattform / Events / Expert-Formate).

---

## Objective

Die aus Phase 20-04 gelieferte 4-Card Bento-Section auf Hero-Level-Qualität ziehen. DS-Typography, DS-Motion, DS-Spacing, saubere Hover-States.

## Context (zu lesen bei Execution)

- `apps/website/components/sections/offering-section.tsx` — aktuelle Variante
- `brand/Generation AI Design System/DESIGN.md` §C Interaction-States, §E Motion
- `brand/Generation AI Design System/preview/card-tool.html` — DS-Card-Referenz
- Hero + SignalGrid als visual quality reference

## Out-of-scope

- Neue Offering-Items oder Umschichten der 4 Cards
- Änderung der Lucide-Icons (Users / BookOpen / Calendar / Mic2)
- Finale Copy
- Deep-Link-Ziele

## Polish-Scope

- Typography: DS-konform (Geist Sans Body, Geist Mono für Labels/Tags falls vorhanden)
- Hover-States: DS §C Interaction-States (scale 1.03 + glow)
- Motion-Easing: `cubic-bezier(0.16,1,0.3,1)` / `--ease-out` / `--dur-normal`
- Spacing: DS-scale
- Card-Radius + Shadow: DS-tokens
- Theme-awareness (light + dark)
- Reduced-motion fallback

## Success Criteria

- [ ] Cards fühlen sich Hero-gleichrangig an
- [ ] Alle Tokens DS-aligned, keine stray Hex
- [ ] Hover + focus-visible States sauber
- [ ] Mobile Layout sauber
- [ ] Reduced-motion fallback sauber
- [ ] Playwright-Baseline grün, Build grün
