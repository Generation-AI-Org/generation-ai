### 2. Signal-Grid Canvas Component (Plan 20.5-02 ✅)
Neue Canvas-Komponente `apps/website/components/ui/signal-grid.tsx` als Core-Landing-Visual-Language. Features:
- 500+ Nodes in pseudo-3D (perspective projection + kinetic depth rotation)
- Cursor-driven Propagation: Hop-1 mit 1 screen-buddy + 1 depth-buddy, Hop-2 mit quadratischem Fade
- Chain-Queue (MAX_CHAINS 2, hop-1 delay 60ms) für fast cursor movement ohne Chaos
- Radial Fade-Overlay für Fokus auf Hero-Content
- Reduced-Motion: statisches Snapshot-Frame, keine Animation

Key commit: `b16e444` feat(20.5-02): signal-grid canvas component.
UAT-Tuning: ~20 Commits (dichte, rotation, propagation, depth, farb-balance, text-readability).

### 3. Hero Rewrite with Signal-Grid (Plan 20.5-03 ✅)
- `<AuroraBackground>` / `<GridBackground>` → `<SignalGrid>` swap.
- Primary-CTA komplett DS-konform: Pill-Radius, hover:scale-1.03, active:scale-0.98, `--accent-glow` shadow, `cubic-bezier(0.16,1,0.3,1)` 300ms easing.
- Text-Backdrop für H1/H2/CTA auf Signal-Grid — Readability ohne Grid zu ersticken.
- `grid-background.tsx` als `@deprecated` markiert (nicht gelöscht, könnte für dezentere Section-Backgrounds wiederverwendet werden).

Key commits: `996f3fb`, `e2c337b`, `dd461e6` (stronger hero text backdrop).
