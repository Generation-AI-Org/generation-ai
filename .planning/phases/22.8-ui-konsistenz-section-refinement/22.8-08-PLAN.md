# Plan 22.8-08 — About Verein-Formular

## Goal

`/about#verein` bekommt ein echtes Formular analog zum Partner-Inquiry-Pattern. Vereins- und Mitmach-Anfragen laufen über Resend an `info@generation-ai.org`, ohne `mailto:` als primären CTA.

## Changes

- Neue Server-Action `submitVereinInquiry` mit Honeypot, Feldvalidierung, Resend-Adminmail und optionaler Bestätigungsmail.
- Neue React-Email-Templates `VereinInquiryEmail` und `VereinInquiryConfirmationEmail`.
- Neue Client-Komponente `AboutVereinForm` als DS-konforme Form-Card im bestehenden Verein-Block.
- Mitmach-CTA springt zum Formular statt den Mailclient zu öffnen.
- About-E2E erweitert um sichtbares Formular, Feldvalidierung und optionalen Submit-Smoke bei vorhandener `RESEND_API_KEY`.

## Must-Haves

- [x] Formular ist im bestehenden Verein-Abschnitt eingebettet.
- [x] Admin-Benachrichtigung geht an `info@generation-ai.org`.
- [x] Kein neuer Live-Service-Write außer Resend beim expliziten Submit.
- [x] Typecheck, Build und About-E2E grün.
